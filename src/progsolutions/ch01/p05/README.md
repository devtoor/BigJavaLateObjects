## P1.5

Write a program that displays your name inside a box on the screen. Do you best to approximate lines with characters such as | - +.

---

Solution:

```java
import static java.lang.System.out;

class P0105 
{
  public static void main(String[] args) 
  {
    out.println("+-----+");
    out.println("| Jay |");
    out.println("+-----+");
  }
}
```
