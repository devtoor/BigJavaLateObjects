## P1.13

Write a program the prints the United States flag, using * and = characters.

---

Solution:

```java
import static java.lang.System.out;

class P0113 
{
  public static void main(String[] args) 
  {
    out.println("**********==================");
    out.println("**********==================");
    out.println("**********==================");
    out.println("**********==================");
    out.println("============================");
    out.println("============================");
    out.println("============================");
  }
}
```
