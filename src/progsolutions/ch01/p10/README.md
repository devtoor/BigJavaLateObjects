## P1.10

Write a program that prints an animal speaking a greeting, similar to (but different from) the following:

```bash
 /\_/\     -----
( ' ' )  / Hello \
(  -  ) <  Junior |
 | | |   \ Coder!/
(__|__)    -----
```

---

Solution:

```java
import static java.lang.System.out;

class P0110 
{
  public static void main(String[] args) 
  {
    out.println("       $$$$$$        ________ ");
    out.println("      $$     $$     |        |");
    out.println("     $  (@)    $$   | Hi     |");
    out.println("   $$          $   <  there! |");
    out.println("      $$     $      |________|");
    out.println("     $    $                   ");
    out.println("    $    $  $$$  $$      $    ");
    out.println("  $$     $     $$  $$  $$$    ");
    out.println("  $      $           $$  $    ");
    out.println("  $$       $      $$     $    ");
    out.println("  $$        $$$$$$      $     ");
    out.println("   $$$                $       ");
    out.println("     $$$$          $$         ");
    out.println("       $$$$$$$$$$$$           ");
  }
}
```
